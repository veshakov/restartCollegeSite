Инструкция по созданию базы данных MySQL на хостинге Sweb


1. На сайте spaceweb.ru

внизу страницы выбрать "Бесплатный хостинг"

и получить его

2. Зарегистрироваться. На указанную почту придет письмо с логином и паролем доступа к панели управления сайтом

3. В пункте "Хостинг"

загружаем все файлы нашего проекта в папку "public_html" обычным перетаскиванием их из проводника Windows

4.Создать базу данных

Имя и пароль сохранить в надежном месте, они будут нужны для получения доступа к базе

Открыть phpMyAdmin для работы с базой

Открыть нашу базу

Создать пустую таблицу

Дать названия столбцам, указать тип данных и их длину

Выбрать таблицу

Выбрать "Вставить" и заполнить поля

Подтвердить

Данные сохранены

Убеждаемся, что в таблице появилась первая строка
