Запись информации в базу данных


1. Для отправки информации на сервер нужно создать форму, на HTML она должна выглядить так (и мы ее создадим на PHP ):

Здесь:
"order.php" - файл, в который отправляется информация из формы на обработку, "id_product" - порядковый номер нашего товара в таблице

Остальные параметры будут объяснены потом

2. Эта форма выглядит как простая кнопка, при нажатии на которую отправляется номер нашего товара для последующей обработки. Эту кнопку с помощью PHP нужно добавить к каждой карточке товара.

type = "hidden" добавляем для того, чтобы сделать номер товара невидимым для покупателя, т.к. это информация для внутреннего пользования

3. Теперь на главной странице у каждого товара появилась кнопка "Купить"

4. Создаем файл-обработчик "order.php"

В результате появляется новая форма заказа для покупателя, в которую он напишет имя и телефон.
А номер товара, т.е. id_product, нам передается из предыдущей страницы командой "$id_product = $_POST['id_product'];"
(в PHP все переменные начинаются со знака "$", поэтому $id_product)

После заполнения полей формы и нажатия кнопки "Оформить заказ" информация оправляется на обработку в файл "confirm.php", который сделаем позже.

5. Добавляем в нашу базу данных новую таблицу, в которую будет заноситься информация о заказах, т.е. контакты покупателя и номер заказываемого товара. Называем ее, например, "prod_order"

6. Даем названия столбцам, нажимаем "Сохранить"

Получаем новую пустую таблицу

6. Теперь создаем файл "confirm.php"

Выбираем на первой странице товар -> "Купить" -> заполняем данные покупателя -> "Оформить заказ"

Если все прошло нормально, то видим надпись "Заказ оформлен"

Проверяем, что запись сохранилась в базе данных
