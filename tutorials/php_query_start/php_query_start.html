Подключение к базе данных и чтение ее содержимого


Внимание: файл "index.php" нужно скопировать в папку "public_html" на хостинге и не забывать при каждом изменении его переписывать

В дальнейшем мы научимся создавать виртуальную копию сервера с базой данных на своем компьютере для отладочных работ

1. Создать файл "index.php" и поставить открывающий и закрывающий теги для вставки кода на серверном языке PHP для работы с базой данный

2. Создать специальный объект PDO для подключения к нашей базе, указав ее имя и пароль (их указывали при создании базы)

3. Сделать запрос на языке SQL для получения всего содержмого таблицы "product"

"SELECT * from product" означает выдать построчно все данные из таблицы

4. Результатом этого запроса будет таблица. Она записывается в память в переменную "$query" и пока нигде не отображается

5. Извлекаем с помощью команды foreach данные из $query строку за строкой и выводим их в содержимое нашей страницы

6. Получаем в браузере все данные в одну строку слитно, без форматирования.

7. На PHP мы можем добавлять любой текст и символы HTML-разметки
